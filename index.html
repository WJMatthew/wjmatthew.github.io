AUTOCORRELATION



Durbin-Watson Test


> lmtest::dwtest(formula = model1, data = data)


    Durbin-Watson test


data:  model1

DW = 0.33192, p-value = 6.346e-10

alternative hypothesis: true autocorrelation is greater than 0


  • We test for serial correlation using the Durbin-Watson test and find that the p-value is less than 0.05, indicating possible serial correlation that needs to be corrected.
  • Cochrane-Orcutt remedy only works when data is an AR(1) stationary process
    • "In other words, taking a first difference of the data results in a stationary process whereby the data has a constant mean, variance and autocorrelation." [1]
